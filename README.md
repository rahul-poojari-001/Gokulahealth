## 📌 Project Overview  

Gokula-Health is an Android application developed for small-scale dairy farmers to digitally manage cattle health, milk production, breeding cycles, and vaccination schedules.  

The application acts as a Digital Health Passport for Cattle, helping farmers improve livestock productivity and reduce financial losses caused by missed vaccinations and improper health tracking.

---

## 🎯 Problem Statement  

Small-scale dairy farmers often miss:
- Heat cycles of cattle
- Vaccination schedules
- Milk yield tracking

Most livestock records are maintained orally, leading to:
- Reduced breeding opportunities
- Disease risks
- Poor productivity
- Income loss

# 📱 Core Features  

## 🐄 Cattle Profile Management
- Register cattle details
- Add cow photo
- Store ear tag ID
- Manage breed and age information

## 🥛 Milk Diary
- Record morning milk yield
- Record evening milk yield
- Track daily production history

## 💉 Vaccination Reminder System
- Automated offline reminders
- FMD vaccination alerts
- Health notification scheduling

## 📊 Yield Analytics
- Milk production trend graphs
- Monthly average yield calculation
- Visual data representation using charts
